# 📊 Data Analysis & Visualization with Python

This repository documents my transition into **Data Science**, focusing on data manipulation, exploratory data analysis (EDA), and graphical representation using industry-standard Python libraries.

---

## 🛠️ Core Tech Stack
* **Pandas:** For robust data cleaning and DataFrame manipulation.
* **Matplotlib & Seaborn:** For creating insightful visualizations (Line, Bar, and Scatter plots).
* **Google Colab:** Cloud-based environment for interactive data storytelling.

---

## 📈 Key Learning Modules

### 1. Data Manipulation (Pandas)
* **Data Loading:** Importing and reading external `.csv` datasets.
* **Filtering & Selection:** Querying specific data points and handling indices.
* **Data Cleaning:** Managing missing values and data type conversions to ensure accuracy.

### 2. Exploratory Data Analysis (EDA)
* **Statistical Summaries:** Using methods like `.describe()` and `.info()` to understand data distribution.
* **Grouping & Aggregation:** Summarizing data to find trends across different categories.

### 3. Data Visualization
* **Trend Analysis:** Plotting line charts to visualize changes over time.
* **Categorical Comparison:** Utilizing bar charts to compare different groups.
* **Customization:** Adjusting figure sizes, labels, and color palettes for better readability.

---

## 🔍 Featured Examples in this Notebook
* **Dataset Overview:** Displaying the first few rows of a dataset to identify patterns.
* **Conditional Plotting:** Creating visuals based on filtered subsets of data.
* **Real-world Practice:** Hands-on exercises involving actual CSV data sources.
